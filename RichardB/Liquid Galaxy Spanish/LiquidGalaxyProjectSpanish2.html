Liquid Galaxy

Un grupo de computadoras que ejecutan Google Earth, Street View y otras aplicaciones panorámicas para crear una experiencia inmersiva.

Learn more »

Visión general

La plataforma Liquid Galaxy utiliza un grupo de computadoras, controladores y pantallas múltiples para brindar una experiencia panorámica envolvente. Comenzando como un proyecto del 20% de Google, Liquid Galaxy ahora se puede encontrar en más de 50 ubicaciones permanentes en todo el mundo, así como en numerosas conferencias y eventos.

Las partes principales de Liquid Galaxy son de código abierto: scripts para sincronizar la vista en instancias de aplicaciones, así como scripts para ajustar el ángulo de visión de un entorno renderizado en 3D (como Google Earth).).

Este sitio web contiene enlaces a repositorios de códigos donde puede encontrar características específicas, guías de instalación e incluso planos para construir su propio marco de pantalla:

  • Guía de inicio rápido para usar la sincronización de vista
  • Liquid Galaxy CÓMO(HOWTO)
  • Uso de 3Dconnexion SpaceNavigator con Google Earth en Linux
  • Diseño mecánico para nuestro marco a medida.

A continuación, también encontrará una lista de problemas conocidos en GitHub y una galería de instalaciones de todo el mundo.

Obtener el codigo

El Liquid Galaxy combina varias aplicaciones para ver contenido, así como bibliotecas, para admitir esas aplicaciones en un sistema correctamente mantenido. El Liquid Galaxy puede ejecutarse en sistemas Linux, Mac y Windows. La evidencia anecdótica muestra que Linux es el más estable.

End Point Corporation ha desarrollado una nueva y extensa base de código de nodos ROS (Robot Operating System) para proporcionar la funcionalidad de Liquid Galaxy. Este sistema se encuentra en desarrollo activo y proporciona un marco organizado para ampliar el sistema.
